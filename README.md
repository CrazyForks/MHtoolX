# MHtoolX v1.4.0  
*A multifunctional interactive mathematical tool / 多功能交互式数学工具*

---

## 📘 Overview | 概述

**MHtoolX v1.4.0** 是一个基于 Python 的命令行交互式数学计算工具，支持32+种数学功能，包括：
- 基础算术、阶乘、质因数分解、质数检测  
- 几何与三角函数绘图（基于 `turtle`）
- 统计分析（均值、方差、标准差、中位数、众数）
- 各类进制转换（十进制、二进制、十六进制）
- 极坐标与笛卡尔坐标转换
- 多项式求解与数值积分
- 矩阵行列式计算与矩阵运算
- 常见数学模型计算
- 自动保存结果与操作历史

该工具提供 **交互式菜单导航** 与 **结果导出功能**，适合教学、科研及学习用途。

## 🚀 How to Run | 使用方法

### 1️⃣ 下载程序
下载 `MHtoolX-v1.4.0.py` 文件

### 2️⃣ 运行程序
```bash
python MHtoolX-v1.4.0.py
```

### 3️⃣ 使用交互命令
程序启动后输入以下命令：
- `menu`：查看分页菜单（共5页）
- `amenu`：查看所有功能列表
- `exit`：退出程序

首次运行时会提示：
- 是否保存操作历史
- 是否启用计算结果保存功能
- 是否启用图像保存功能

---

## 💾 Data & History Management | 数据与历史记录管理

程序自动在运行目录下生成以下配置文件：

| 文件名 | 功能说明 |
|--------|-----------|
| `result.json` | 保存历史计算结果 |
| `history.json` | 保存用户操作记录 |
| `number saved.json` | 已保存结果计数（可选启用） |
| `config.json` | 程序配置设置（历史保存、结果保存、图像保存等） |

**管理命令**：
- `er`：导出历史计算结果到 `result.txt`
- `eh`：导出操作历史到 `history.txt`
- `cr`：清空历史计算结果
- `ch`：清空操作历史
